# Brand

**Brand name: ProductPicks**

**Date established: 2026-07-11**

This name was deliberately chosen to be generic and niche-agnostic, so the site can expand beyond its original pickleball-gear content vertical (see `niche.md`) without requiring another rebrand. Future daily automation runs should use "ProductPicks" consistently if they ever need to reference the site's own name (individual posts usually won't need to).

**Update, 2026-07-12:** that planned expansion has begun — the site now spans multiple categories (Sports, Home & Kitchen, Outdoor & Recreation, and more over time). See `categories.md` for the active category list and taxonomy.
